  Link to our service: https://yoplanning.com  
  Link to support and documentation: https://en.yoplanning.support  
  Link to terms and privacy: https://yoplanning.pro/privacy-terms

== Installation ==

FR : 
1. Déposer le dossier `yoplanning-booking-engine` vers le dossier `/wp-content/plugins/`.
2. Activer le plugin dans la rubrique "Plugins" du menu de WordPress.

EN : 
1. Upload the `yoplanning-booking-engine` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
